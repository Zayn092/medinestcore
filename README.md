# MediNestCore

## Problem
Managing medical prescriptions manually is insecure, error-prone, and difficult to scale.

## Solution
MediNestCore is a web-based prescription management system that securely stores and manages medical records using role-based authentication.

## Features
- User authentication
- Secure prescription storage
- Role-based access
- CRUD operations
- Structured database design

## Tech Stack
- PHP
- MySQL
- JavaScript
- HTML, CSS, Bootstrap

## Setup Instructions
1. Clone the repository
2. Import the database file into MySQL
3. Configure database connection
4. Run the project on a local server (XAMPP)
